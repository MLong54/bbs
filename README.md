# flask_bbs
基于python flask 的个人论坛

* 论坛实现了注册登录，发帖，评论，私信，用户个人信息修改等功能
* 使用`Mysql`数据库以及`Flask_SQLAlchemy`框架实现数据持久化与存储
* 论坛使用`gunicorn`来管理项目，使用`nginx`进行反向代理
* 并通过编写`bash`脚本，实现一键部署项目环境
